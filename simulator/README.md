# DRYSUITS Graphite α カラーシミュレーター

DRYSUITS Graphite αの各パーツの色を変更できるWebベースのカラーシミュレーターです。

## 機能

- A〜Dパーツの個別色変更
- 元画像の陰影・立体感を保持した高品質な色変換
- Bパーツの黒ラインを黒固定で保持
- 品番付きカラーパレット（16色）
- PNG形式での配色保存
- URLパラメータによる状態保存・共有
- レスポンシブ対応（スマホ閲覧可能）

## ローカルで実行する方法

### 方法1: 簡易HTTPサーバー（推奨）

Python 3がインストールされている場合：

```bash
cd simulator
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` にアクセスしてください。

### 方法2: Node.jsのhttp-server

```bash
npx http-server simulator
```

### 方法3: VS CodeのLive Server拡張機能

1. Live Server拡張機能をインストール
2. `index.html` を右クリックして「Open with Live Server」を選択

**注意:** 画像読み込みの制約により、直接ファイルを開く（`file://`）場合はCORSエラーが発生する可能性があります。HTTPサーバーの使用を推奨します。

## 使用方法

1. 各パーツ（A〜D）のカラーパレットから色を選択
2. プレビューがリアルタイムで更新されます
3. 「初期状態に戻す」ボタンでデフォルト配色に戻せます
4. 「PNGで保存」ボタンで現在の配色を画像として保存できます

## URLパラメータによる状態保存

現在の配色状態はURLに自動保存されます。例：

```
?A=GY&B=WH&C=BE&D=DG
```

このURLを共有することで、他の人も同じ配色を確認できます。

## カラーパレットの追加・変更方法

`js/palette.js` を編集します。

### 新しい色を追加する場合

`COLOR_PALETTE` 配列に以下の形式で追加してください：

```javascript
{ code: '品番', name: '色名', hex: '#RRGGBB' }
```

例：

```javascript
{ code: 'CY', name: 'Cyan', hex: '#00ffff' }
```

### 既存の色を変更する場合

該当する色の `hex` 値を変更してください。

## 色変換ロジックの説明

`js/colorizer.js` で実装されている色変換アルゴリズムは以下の手順で動作します：

1. **輝度計算**: 元ピクセルの輝度（明るさ）をITU-R BT.709係数で計算
   ```
   L = 0.2126*R + 0.7152*G + 0.0722*B
   ```

2. **目標色のHSL変換**: 選択された色の色相(H)と彩度(S)を取得

3. **輝度保持の色変換**: 元の輝度と目標色の色相・彩度を組み合わせて新しいRGBを生成

4. **黒ライン保護**: Bパーツの暗色ピクセル（RGB各値が50未満）は色変換をスキップ

これにより、元画像の陰影やテクスチャを保持したまま、自然な色変更が可能です。

## パーツ定義

| パーツ | 説明 |
|--------|------|
| Aパーツ | 胸下〜胴体〜股下付近のグレー部分 |
| Bパーツ | 肩〜胸上の白いパーツ部分（黒ライン固定） |
| Cパーツ | 左右の腕のベージュ部分 |
| Dパーツ | すね付近のグレー部分 |
