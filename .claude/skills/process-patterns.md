# Process Patterns Skill

柄テクスチャの高密度化・シームレス化処理を一括実行するスキルです。

## Usage

```
/process-patterns
```

## Description

全ての柄テクスチャ（BCF、ASH、LEP、SEA、BLEP）を処理し、シームレスなパターンに変換します。

## Instructions

ユーザーがこのスキルを呼び出したら、以下を実行してください：

### 実行コマンド

```bash
cd /Users/takahashikentarou/Desktop/Topclaude/Topファイル/blala
python3 カラー一覧/process_patterns.py
```

### 処理内容

| パターン | 処理タイプ | タイリング係数 | 縮小率 |
|----------|------------|----------------|--------|
| BCF | 繊細化 | 2.0 | 50% |
| ASH | 繊細化 | 1.5 | 67% |
| LEP | 標準 | - | - |
| SEA | 標準 | - | - |
| BLEP | 標準 | - | - |

**繊細化処理（BCF, ASH）:**
1. 元画像の中央から4096x4096を切り出し
2. タイリング＆ダウンスケールで密度向上
3. シームレス化（ブレンド幅32px）
4. 512x512で出力

**標準処理（LEP, SEA, BLEP）:**
1. 元画像の中央から512x512を切り出し
2. シームレス化（ブレンド幅24px）
3. 512x512で出力

### 出力先

`カラー一覧/一覧/processed/` ディレクトリに `{CODE}_processed_thumb.png` として保存
